Project B - CSV Mini Database & Query Engine
Project B is a lightweight implementation of a CSV-backed mini database and query engine. It is designed to ingest CSV files, store tabular data efficiently in memory, and support query operations similar to a simple database engine.

Project Description
This project builds a compact database system for structured CSV data. It focuses on:

Parsing CSV files and managing rows and columns.
Supporting query operations such as selection, projection, filtering, sorting, and aggregation.
Providing a simple query interface for extracting and transforming CSV data.
Maintaining a clean, modular implementation suitable for learning database engine fundamentals.
Goals
Enable users to load CSV tables and execute expressive queries.
Implement core query engine components without relying on a full database system.
Demonstrate understanding of data parsing, storage, and query execution.
Deliver a practical tool for working with CSV datasets in a lightweight, self-contained format.
Usage
Load a CSV file into the mini database.
Define query expressions to filter, sort, and retrieve data.
Execute queries and inspect the results.
